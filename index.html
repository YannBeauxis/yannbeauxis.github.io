Metwork

MetWork est une plateforme web open source de calcul en chimie des substances naturelles.

J’ai développé la solution en Python avec Django et la JSON REST API pour le backend et EmberJS pour le frontend. La charge de calcul en chimie analytique est répartie sur 6 serveurs via un système de task management utilisant Celery et RabbitMQ.

Le projet a fait l’objet d’une publication dans la revue Bioinformatics .

Lien vers la plateforme : https://metwork.pharmacie.parisdescartes.fr
Sur GitHub : https://github.com/metwork-project

  • Client: Université de Paris
  • Date: janvier à aujourd'hui
